# Philharmonic Workspace

Generic workflow orchestration infrastructure. 25 Rust crates,
three deployment binaries, end-to-end encrypted connector
pipeline with post-quantum cryptography.

This book contains the design documentation and reference
material. For development conventions, see
[CONTRIBUTING.md](https://github.com/metastable-void/philharmonic-workspace/blob/main/CONTRIBUTING.md)
in the repository root.
