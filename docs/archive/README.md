## Archive

Point-in-time snapshots of workspace primary docs that have since been
trimmed. Format: `YYYY-MM-DD-<topic>.md`.

Trim driver is usually token cost: every push-time LLM summarizer
reads `README.md` / `docs/ROADMAP.md` / friends, and large duplicated
content is paid for by every push. When a primary doc is trimmed, the
removed content is preserved here verbatim with an explanatory header.

These files are not load-bearing for current development — they
exist for historical reference only. The authoritative current state
lives in the file the snapshot was taken from.
